### 🎴 精美卡牌展示

## 🔧 技术架构

### 前端技术栈
- **纯前端实现**：HTML5 + CSS3 + JavaScript ES6+
- **CSS3动画**：3D Transform + Animation
- **响应式设计**：Flexbox + Grid + Clamp()函数
- **模块化架构**：清晰的代码组织结构

### 核心算法
- **精确时间同步**：基于performance.timing的动画同步
- **智能卡牌切换**：在侧面位置无缝切换，避免变牌
- **Fisher-Yates洗牌**：确保真正的随机性
- **能量分析引擎**：分析牌面能量和主题关联

### 性能优化
- **图片懒加载**：智能预加载和缓存策略
- **移动端适配**：网络状况感知和设备性能优化
- **内存管理**：智能缓存清理和内存优化
- **动画优化**：GPU加速和60FPS性能保证
