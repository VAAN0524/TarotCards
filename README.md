# 塔罗牌占卜网站

一个神秘而优雅的塔罗牌占卜网站，提供22张大阿卡那牌的展示和专业的占卜解读服务。

## 🌟 特性

### 🎴 塔罗牌展示
- 22张大阿卡那牌自动旋转展示
- 优雅的3D翻转动画效果
- 随机展示系统，避免重复
- 响应式设计，适配各种设备

### 🔮 占卜功能
- **多种占卜类型**：
  - 三牌阵（过去-现在-未来）
  - 凯尔特十字（全面分析）
  - 决策指引（选项分析）
  - 单张指引（今日运势）

- **智能解读系统**：
  - 根据问题类型个性化解读
  - 支持正位/逆位含义
  - 涵盖爱情、事业、健康、财运四个维度
  - 能量分析和综合建议

### 🎨 设计特色
- 神秘的紫色渐变主题
- 金色艺术体标题和按钮
- 呼吸灯和脉冲动画效果
- 优雅的悬停交互效果

## 🚀 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/VAAN0524/TarotCards.git
   cd TarotCards
   ```

2. **直接使用**
   - 打开 `index.html` 查看塔罗牌展示
   - 打开 `tarot-reading.html` 开始占卜

3. **本地服务器**（可选）
   ```bash
   # 使用Python
   python -m http.server 8000

   # 使用Node.js
   npx serve .

   # 然后访问 http://localhost:8000
   ```

## 🔧 自定义配置

### 修改卡牌图片
将你的22张大阿卡那牌图片放在 `images/` 目录下，命名格式：
```
{id}. {中文名} ({英文名}).png
```

### 调整动画速度
在 `style.css` 中修改：
```css
.card {
    animation: autoSpin 8s linear infinite; /* 修改8s调整速度 */
}
```

### 自定义颜色主题
在 `style.css` 中修改CSS变量：
```css
:root {
    --primary-color: #d4af37;  /* 金色 */
    --bg-gradient: linear-gradient(...); /* 背景渐变 */
}
```
